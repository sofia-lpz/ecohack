Calculator Endpoint

GET /api/calculator/{fertilizer}/{nitrogen}

Fertilizer float

Required: Kilograms of fertilizer


Nitrogen int

Required: Nitrogen percentage in the fertilizer per kilogram


Expected Response:

status: "OK"
data: {float}




Year Emission Endpoint

GET /api/historic/{year}

Year int

Required: Year of query in format like {2024}


Expected Response:

status: "OK"
data: {float}




Country Year Emission Endpoint

GET /api/historic/{country}/{year}

Country string

Required: Name of the country


Year int

Required: Year of query in format like {2024}


Expected Response:

status: "OK"
data: {float}




Year Range Emission Endpoint

GET /api/historic/range/{yearstart}/{yearend}

Year Start int

Required: Start year of query in format like {2020}


Year End int

Required: End year of query in format like {2024}


Expected Response:

status: "OK"
data: {list of float}
